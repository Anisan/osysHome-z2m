# Zigbee2MQTT (`z2m`)

The `z2m` documentation is split into two main documents:

- [User Guide](USER_GUIDE.md)
- [Technical Reference](TECHNICAL_REFERENCE.md)

## What to open

If you want to quickly connect Zigbee2MQTT, view devices, set links, and start controlling devices from osysHome, open [User Guide](USER_GUIDE.md).

If you need internals (MQTT flow, queue worker, converters, REST endpoints, WebSocket events, data model), open [Technical Reference](TECHNICAL_REFERENCE.md).
