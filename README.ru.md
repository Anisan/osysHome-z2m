# z2m - Интеграция Zigbee2MQTT

![z2m Icon](static/z2m.png)

Интеграция на основе MQTT с мостом Zigbee2MQTT для управления устройствами Zigbee через протокол MQTT.

## Описание

Модуль `z2m` предоставляет интеграцию с мостом Zigbee2MQTT для платформы osysHome. Он обеспечивает обнаружение, управление и мониторинг устройств Zigbee через протокол MQTT.

## Основные возможности

- ✅ **Интеграция MQTT**: Коммуникация устройств на основе MQTT
- ✅ **Обнаружение устройств**: Автоматическое обнаружение устройств Zigbee
- ✅ **Управление свойствами**: Управление свойствами устройств
- ✅ **Связывание свойств**: Связывание свойств устройств со свойствами объектов
- ✅ **Мониторинг батареи**: Мониторинг уровней батареи устройств
- ✅ **Отслеживание доступности**: Отслеживание доступности устройств
- ✅ **Интеграция поиска**: Поиск устройств и свойств
- ✅ **Поддержка виджетов**: Виджет панели управления со статистикой устройств

## Панель администратора

Модуль предоставляет административный интерфейс для:
- Просмотра устройств Zigbee
- Настройки свойств устройств
- Связывания свойств с объектами
- Мониторинга статуса устройств

## Конфигурация

- **MQTT брокер**: Настройки подключения к MQTT брокеру
- **Топик**: Префикс MQTT топика (по умолчанию: zigbee2mqtt)
- **Управление устройствами**: Настройка свойств устройств и связей

## Использование

### Добавление устройства Zigbee

1. Убедитесь, что Zigbee2MQTT запущен и подключен к MQTT брокеру
2. Перейдите в модуль z2m
3. Устройства обнаруживаются автоматически через MQTT
4. Настройте свойства устройств
5. Свяжите свойства со свойствами объектов

## Технические детали

- **Протокол**: MQTT
- **Мост**: Zigbee2MQTT
- **Типы устройств**: Все типы устройств Zigbee, поддерживаемые Zigbee2MQTT
- **Сопоставление свойств**: Автоматическое сопоставление свойств из Zigbee2MQTT
